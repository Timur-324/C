Ниже приведено подробное описание трёх связанных тем: условный оператор (конструкции выбора), логические операции и выражения, а также тернарный оператор.

─────────────────────────────  
1. Условный оператор (if, if-else, switch)

• if/else  
 – Позволяет выполнить блок кода только если условие истинно (true).  
 – Базовый синтаксис:
  if (условие) {
   // действия, если условие истинно
  } else {
   // действия, если условие ложно
  }
 – В языке C (и подобных) условие – это выражение, которое приводится к целому типу: 0 считается false, отличное от нуля – true.

• Последовательные условия  
 – Можно использовать конструкцию else if для проверки нескольких вариантов:
  if (условие1) {
   // блок 1
  } else if (условие2) {
   // блок 2
  } else {
   // блок по умолчанию
  }

• Switch (переключатель)  
 – Используется для множественного выбора на основе значения одного выражения.
 – Синтаксис:
  switch(выражение) {
   case значение1:
    // действия
    break;
   case значение2:
    // действия
    break;
   default:
    // действия по умолчанию
    break;
  }
 – Обычно применяется, когда вариантов выбора много и они зависят от конкретных значений (целочисленных или перечислимых типов).

─────────────────────────────  
2. Логические операции и выражения

Логические операции позволяют комбинировать или инвертировать логические значения (true/false) и часто используются в условиях условных операторов.

• Основные логические операторы:
 – Оператор И (AND): &&  
  • Выражение A && B истинно только если оба A и B истинны.
  • Использует короткое замыкание: если A ложно, B даже не вычисляется.
 – Оператор ИЛИ (OR): ||  
  • Выражение A || B истинно если хотя бы одно из выражений истинно.
  • Также использует короткое замыкание: если A истинно, B не вычисляется.
 – Логическое НЕ (NOT): !  
  • Инвертирует логическое значение: !true становится false, и наоборот.

• Логические выражения  
 – Состоят из операндов (выражений или переменных) и логических операторов.  
 – Например:
  if ((a > 0) && (b < 10)) { ... }
 – Приоритет операторов:  
  • ! имеет более высокий приоритет, чем && и ||.  
  • && имеет более высокий приоритет, чем ||.  
 – Скобки используются для явного управления порядком вычислений.

• Применение  
 – Логические операции часто комбинируют условия в if, while, for и других конструкциях, позволяя задавать сложные критерии выполнения кода.

─────────────────────────────  
3. Тернарный оператор (оператор "?:")

Тернарный оператор – компактная форма записи условного оператора, позволяющая выбрать одно из двух значений на основе логического условия.

• Синтаксис:
  условие ? выражение1 : выражение2
 – Если условие истинно, результатом всей конструкции становится выражение1; если ложно – выражение2.

• Пример:
  int max = (a > b) ? a : b;
  • Если a больше b, переменной max присваивается значение a, иначе – значение b.

• Особенности:
 – Тернарный оператор является выражением, то есть возвращает значение и может использоваться в более сложных выражениях.
 – Он часто применяется там, где требуется компактная запись, хотя чрезмерное его использование может снизить читаемость кода.
 – Важно помнить о правильном использовании скобок, если тернарный оператор находится внутри других выражений, чтобы избежать неоднозначности в порядке вычисления.

─────────────────────────────  
Общий вывод

• Условные операторы (if/else, switch) позволяют ветвить выполнение программы в зависимости от условий.  
• Логические операции (&&, ||, !) составляют основу формирования условий: они комбинируют, проверяют и изменяют логические значения.  
• Тернарный оператор – это лаконичный способ выбора между двумя выражениями на основе логического условия.

Эти механизмы часто применяются вместе: логические выражения формируют условие для if/else, а тернарный оператор обеспечивает компактную запись простых выборов. Понимание работы и особенностей каждого из них помогает писать более понятный, эффективный и поддерживаемый код.
