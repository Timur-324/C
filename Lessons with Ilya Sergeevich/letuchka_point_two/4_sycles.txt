Ниже описаны операторы циклов и механизмы управления выполнением их тела, которые позволяют реализовывать повторяющиеся действия и гибко управлять потоком исполнения.

─────────────────────────────  
1. Операторы циклов

В языке C для организации повторения команд используются три основных оператора цикла:

• for  
 – Обычно применяется, когда известны начальное значение, условие продолжения и шаг изменения счетчика.  
 – Синтаксис:  
  for (инициализация; условие; шаг) {  
   // тело цикла  
  }

• while  
 – Выполняет тело цикла, пока условие истинно.  
 – Синтаксис:  
  while (условие) {  
   // тело цикла  
  }

• do…while  
 – Гарантирует хотя бы одно выполнение тела цикла, затем проверяет условие для повторения.  
 – Синтаксис:  
  do {  
   // тело цикла  
  } while (условие);

─────────────────────────────  
2. Управление выполнением тела операторов цикла

Для изменения стандартного порядка итераций цикла используются следующие механизмы:

• break  
 – Прерывает выполнение цикла и передает управление на оператор, следующий за циклом.  
 – Часто применяется для немедленного выхода из цикла, когда выполнено определённое условие.  
  Пример:  
   for (int i = 0; i < n; i++) {  
    if (условие) {  
     break; // выход из цикла  
    }  
    // остальной код  
   }

• continue  
 – Прерывает текущую итерацию цикла и переходит к следующей итерации.  
 – Полезен, если нужно пропустить некоторые шаги внутри цикла и сразу перейти к обновлению счетчика или проверке условия.  
  Пример:  
   while (условие) {  
    // код до continue  
    if (другое условие) {  
     continue; // переход к следующей итерации  
    }  
    // код, который не выполнится при continue  
   }

• Возврат из функции (return)  
 – Хотя не относится непосредственно к циклам, оператор return может использоваться внутри цикла для прекращения не только текущей итерации, но и всей функции.

• goto  
 – Позволяет безусловно перейти к указанной метке в коде.  
 – Использование goto внутри цикла может перескакивать как на начало, так и вне цикла, но зачастую ведет к ухудшению читаемости и не рекомендуется, если есть более структурированные альтернативы.

─────────────────────────────  
3. Примеры применения

Пример с использованием for и break/continue:

  #include <stdio.h>
  
  int main(void) {
   int n = 10;
   for (int i = 0; i < n; i++) {
    if (i == 3) {
     continue;  // пропускаем вывод числа 3
    }
    if (i == 8) {
     break;     // досрочно завершаем цикл, когда i равно 8
    }
    printf("%d ", i);
   }
   return 0;
  }
  
  // Результатом будет: 0 1 2 4 5 6 7  

Пример с do…while:

  #include <stdio.h>
  
  int main(void) {
   int count = 0;
   do {
    printf("Iteration %d\n", count);
    count++;
   } while (count < 5);
   return 0;
  }

─────────────────────────────  
Заключение

• Операторы циклов (for, while, do…while) позволяют организовать повторение команд, что важно для реализации множества алгоритмов.  
• Управляющие операторы внутри цикла (break, continue) дают возможность гибко изменять стандартное поведение цикла, обеспечивая более точный контроль над выполнением тела.  
• Использование данных операторов способствует созданию эффективного и читаемого кода при условии разумного их применения.

Это базовые механизмы, помогающие управлять потоками исполнения в циклических конструкциях. Понимание их работы позволяет избежать ошибок логики и сделать код более понятным и поддерживаемым.
