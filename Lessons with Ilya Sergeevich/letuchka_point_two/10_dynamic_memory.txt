Динамическое выделение памяти в языке C позволяет запрашивать блоки памяти во время выполнения программы, когда заранее невозможно точно определить необходимый размер. Это дает гибкость при работе с массивами переменной длины, динамическими структурами и другими задачами, где объем необходимой памяти не известен на этапе компиляции.

─────────────────────────────  
1. Функция malloc

• Назначение: Выделяет блок памяти заданного размера в байтах.  
• Синтаксис:  
  void *malloc(size_t size);  
• Особенности:
  – Если память успешно выделена, функция возвращает указатель на начало блока; если нет – возвращается NULL.
  – Содержимое выделенной памяти не инициализируется, оно может содержать произвольные данные ("мусор").

• Пример:
  #include <stdlib.h>
  int *array = (int *)malloc(10 * sizeof(int));
  if (array == NULL) {
   // обработка ошибки выделения памяти
  }

─────────────────────────────  
2. Функция calloc

• Назначение: Выделяет память для массива элементов заданного количества и размера, при этом инициализирует все байты нулевыми значениями.  
• Синтаксис:  
  void *calloc(size_t num, size_t size);  
• Отличия от malloc:
  – calloc принимает два аргумента: количество элементов и размер каждого элемента.
  – В отличие от malloc, calloc гарантирует, что вся память будет инициализирована нулями.

• Пример:
  #include <stdlib.h>
  int *array = (int *)calloc(10, sizeof(int));
  if (array == NULL) {
   // обработка ошибки
  }

─────────────────────────────  
3. Функция realloc

• Назначение: Изменяет размер ранее выделенного блока памяти.  
• Синтаксис:  
  void *realloc(void *ptr, size_t new_size);  
• Особенности:
  – Если ptr равен NULL, то realloc работает как malloc.
  – Если new_size равен 0 и ptr не равен NULL, то поведение может быть различным (обычно блок памяти освобождается, а функция возвращает NULL).
  – Если новая область памяти располагается по другому адресу, old данные копируются в новую область.

• Пример:
  #include <stdlib.h>
  int *array = (int *)malloc(10 * sizeof(int));
  // Изменяем размер на 20 элементов
  int *new_array = (int *)realloc(array, 20 * sizeof(int));
  if (new_array == NULL) {
   // обработка ошибки (старый блок по-прежнему действителен)
  } else {
   array = new_array;
  }

─────────────────────────────  
4. Функция free

• Назначение: Освобождает ранее выделенный блок памяти, чтобы он мог быть использован повторно системой.  
• Синтаксис:  
  void free(void *ptr);  
• Важно помнить:
  – После освобождения указатель остается валидным, но указывает на неиспользуемую область памяти (так называемый "dangling pointer").  
  – Следует стараться занулять указатель после освобождения: ptr = NULL.

• Пример:
  #include <stdlib.h>
  free(array);
  array = NULL;

─────────────────────────────  
5. Оператор sizeof

• Назначение: Определяет размер типа или переменной в байтах во время компиляции.  
• Синтаксис:
  sizeof(тип) или sizeof(переменная)  
• Применение:
  – Помогает вычислить, сколько памяти требуется для хранения конкретного типа данных.
  – Часто используется в функциях malloc, calloc и realloc для корректного расчета объема памяти.

• Пример:
  #include <stdio.h>
  printf("Размер типа int: %zu байт\n", sizeof(int));

─────────────────────────────  
Заключение

• malloc – выделяет неинициализированный блок памяти заданного размера.
• calloc – выделяет и инициализирует нулевыми значениями память для массива.
• realloc – изменяет размер уже выделенного блока памяти.
• free – освобождает ранее выделенный блок памяти.
• sizeof – возвращает размер типа или переменной в байтах, что важно для корректного выделения памяти.

Правильное использование этих функций позволяет эффективно управлять памятью в программе на C и предотвращать утечки памяти или прочие ошибки, связанные с динамическим выделением памяти.
